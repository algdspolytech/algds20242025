>Вариант 20: Поиск в глубину. Граф задан списком смежности (2)
Граф задан в файле списком смежности. Заданы две вершины графа. С помощью алгоритма поиска в глубину
узнайте, существует ли путь между ними.

Примечание: В тесте я рассматривал ориентированный граф.
Чтобы рассматривать неориентированный граф, нужно в функции - void addEdge(Graph* graph, int src, int dest), раскомментировать строку - graph->adjList[dest][src] = 1; которая добавляет обратное направление ребра.
